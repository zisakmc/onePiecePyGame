# One Piece game

Just do whatever you like

requirement:

- pygame

```
pip install pygame
```

## To run

```
python play.py
```

#### control

| Action    | Movement   |
| --------- | ---------- |
| left key  | move left  |
| h         | move left  |
| right key | move right |
| l         | move right |
